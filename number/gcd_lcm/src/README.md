# GCD, LCM
2個以上の最小公倍数、最大公約数を求める。

## 計算量
整数$A_1, A_2, \dots, A_N$に対して、$O(N \log{A_{max}})$。
ただし、$A_{max}$は$A_1, A_2, \dots, A_N$の最大値。
